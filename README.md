<h1>IBSayin'</h1>

<p>IBSayin' is the first Meteor App I've created. It's a simple app where you can post quotes and images. You can also view the posts of others.</p>

<p>Last Edited - May 2016</p>

<h4>Instructions</h4>

<ul>
  <li>Clone Repo</li>
  <li>cd into repo and npm install</li>
  <li>meteor run</li>
</ul>
